# dbs-application
A simple full-stack application to show the power of sql

# How to run

Assuming you have Python installed on your computer

## Windows

```shell
py -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
py app.py
```

## Unix

```shell
py -m venv venv
venv\Lib\activate
pip install -r requirements.txt
py app.py
```
